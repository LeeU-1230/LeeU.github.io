JavaScript – Object 資料複製

發表於 2020-12-12 更新於 2020-12-25 分類於 Javascript ， Object ， 資料複製 Disqus：
關於淺層複製、深層複製

淺層複製方法: Object.assign

複製到第一層物件的屬性，若有第二層以上的資料會與舊物件一起共用同一塊記憶體。

淺層複製方法: …其餘運算子

深層複製方法: JSON

複製建立全新的物件。
JSON 方法限制:純資料的物件可行，有Function、Set、Map、undefined等型態會失效。

深層複製方法: $.extend
