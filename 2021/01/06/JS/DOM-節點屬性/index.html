JavaScript – DOM 節點屬性

使用 DOM 節點物件的屬性。

Node.nodeName

取得節點名稱。


Node.nodeValue

用來取得 text, comment 和 CDATA 節點的內容。


Element.innerHTML

修改/取得 一個元素節點中的 HTML 內容。


Node.textContent

和 innerHTML 用法類似，修改/取得 一個元素節點中的 HTML 內容。
不過 textContent 會將 HTML 標籤去除，將 HTML 特殊符號自動轉成字元實體。


Element.outerHTML

修改/取得 一個元素節點中的 HTML 內容，會返回包括節點本身的 HTML 內容。
